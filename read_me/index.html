A) HTML Structure - top

Navigation

The menu is set using id tags each should be unique and link to each section in the content.
You can change the speed and style of scrolling in the file style > js> scripts.js. Scroll down until you see - Navigation and along the code until you see: animationTime:1300,easing:["easeInOutBack"]}; Lowering the time speeds up the scrolling, and easing is the style in which the page scrolls. More effects can be found here - http://gsgd.co.uk/sandbox/jquery/easing/

You can also link to any section of the site using the class "visualNav" in your link along with the id of the section that you want to scroll to.
e.g <a href="#team" class="visualNav">fascinating</a> kind of art.

Home Slider
The sections are controlled by a slider called Awkward (link below).
Pictures sizes are: 830px x 470px. You can add images, video and your own html to this slider.

Portfolio

C) JavaScript & Php - top

This theme uses javascript and php.

  1. Awkward Slider
    Used for the home page image slider. You don't need to do anything with the coding, but if you do you can find everything in the js folder. To add your images, size them 830px x 470px and place them into your images folder. The html has been commented for you. (link in credits).
  2. Team Pictures Overlay
    Sliding Boxes Jquery is used for the fade efect which reveals text of your choice. The script can be edited in style > js > scripts.js, commented - 'Team Pics Overlay'.
  3. Pretty Photo is used for the portfolio section. "prettyPhoto is a jQuery lightbox clone. Not only does it support images, it also support for videos, flash, YouTube, iframes and ajax." (link in credits).
  4. LightBox Effect
    <a href="style/images/work/large.jpg" rel="prettyPhoto[portfolio]">view</a></p> </div><img src="style/images/work/art.jpg" width="269" height="160" alt="" class="portimg" />
    1: large.jpg is your large pop-up image and art.jpg is your thumbnail. Simply add your own graphics and place them in the folder > images > work.
    The portfolio hover opacity can be edited in style > js >scripts.js, commented 'Zoom Portfolio Hover'.
  5. TipTip Jquery
    Used for the hover tool tip effect on text links.
    To get this to work simply add a title tag to your link and:
    a) class="tooltip" - for a hover above tool tip
